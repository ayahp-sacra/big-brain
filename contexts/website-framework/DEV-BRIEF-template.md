## Key Requirements

### Accessibility
- WCAG 2.1 AA compliance
- Minimum body font size: 17–18px
- All color combinations must pass AA contrast ratio
- All images have alt text
- All forms have proper labels

### SEO
- Unique meta title + description on every page
- Open Graph tags (title, description, image)
- Canonical URLs
- [LocalBusiness / other] structured data schema where applicable
- Sitemap.xml generated and submitted

### Performance
- Target Lighthouse score: 90+ on all pages
- Images optimized and served in modern formats (WebP)
- No unused scripts or third-party bloat

### Mobile
- Mobile-first build
- Fully responsive from 375px up
- Tap targets minimum 44×44px
