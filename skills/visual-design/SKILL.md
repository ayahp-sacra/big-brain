# Visual Design Excellence

The default output of AI-assisted design is visually safe, predictable, and forgettable. This skill exists to prevent that. Good design is intentional, committed, and distinctive — it makes a clear decision about who the client is and holds that decision across every element. "Clean and professional" is not a design direction. This skill gives you the tools to arrive at one.

---

## Step 0: Design Intake (Mandatory Before Any Design Work)

Never start designing without answering these four questions. If the client hasn't provided answers, infer them from context — but commit to an answer before proceeding.

**1. What is the ONE emotion this design must evoke?**
Not a list. One. Examples: *trust*, *desire*, *excitement*, *calm*, *exclusivity*, *belonging*, *urgency*, *wonder*.

**2. What is the ONE action the user must take?**
Not a list. One. Examples: *book a call*, *buy now*, *sign up*, *contact us*, *get a quote*. Everything in the design serves this action.

**3. What aesthetic family does this belong to?**
Choose from the Aesthetic Library below. Name it explicitly. If it falls between two, pick the dominant one and note the secondary influence. Do not proceed without a named aesthetic.

**4. What does this brand explicitly NOT look like?**
Name 2–3 things. Examples: *not corporate and cold*, *not playful and childish*, *not sparse and minimal*, *not loud and cluttered*. Constraints are creative tools. Use them.

## Aesthetic Library

Each aesthetic has rules. When you commit to one, follow its rules — don't blend randomly. Strategic influence from a second aesthetic is allowed; accidental hybrid is not.

---

### 1. Swiss / International Typographic Style
**Emotion:** Authority, clarity, confidence
**Best for:** B2B, professional services, finance, healthcare, tech

**Type:** Grotesque sans-serif (Helvetica, Aktiv Grotesk, Neue Haas Grotesk). Grid-locked. Flush left, ragged right. Type size differentials are dramatic — headlines dominate. Body text is small and dense.
**Color:** Near-monochrome. Black, white, one pure accent (red, yellow, or blue — not a tint). Never gradients.
**Layout:** Rigid grid. Asymmetric columns. Content blocks aligned to a strict baseline grid. Negative space is deliberate and structural.
**Imagery:** Black-and-white photography or none. Diagrams. Never decorative.
**Avoid:** Serif body text, rounded corners, gradients, centered alignment, decorative illustration.

---

### 2. Editorial / Magazine
**Emotion:** Curiosity, sophistication, engagement
**Best for:** Lifestyle brands, culture, fashion, food, media, agencies

**Type:** Mix a serif display with a grotesque body — contrast is the point. Headlines are large, expressive, sometimes cropped. Captions exist. Pull quotes break the grid.
**Color:** Restrained palette — usually 2–3 colors, often including black and an unexpected accent. Let photography carry visual weight.
**Layout:** Breaking the grid intentionally. Overlapping elements. Variable column widths. Sections feel like different pages in a magazine.
**Imagery:** High-quality editorial photography. Full-bleed. Cropped tight for drama.
**Avoid:** Generic stock photography, uniform section heights, symmetrical layouts, everything the same size.

---

### 3. Brutalist Web
**Emotion:** Boldness, rawness, confidence, anti-establishment
**Best for:** Creative studios, artists, musicians, fashion brands, provocative brands

**Type:** Big. Bold. Often a single typeface pushed to extremes — 200px headlines, tight letter-spacing, or wide tracking. Monospace as body text is on-brand. Type can break containment.
**Color:** High contrast. Black and white with one aggressive accent, or an intentionally jarring palette. Color is used structurally, not decoratively.
**Layout:** Visible structure. Elements can be misaligned by design. Borders visible. Asymmetry is extreme. Scrolling is part of the experience.
**Imagery:** Raw, unretouched, or deliberately lo-fi. Or none.
**Avoid:** Rounded corners, soft shadows, gradients, stock photography, polished illustrations, conventional nav patterns.

---

### 4. Luxury Minimal
**Emotion:** Exclusivity, calm, aspiration, restraint
**Best for:** High-end services, jewelry, architecture, premium skincare, private members, luxury real estate

**Type:** Refined serif (Canela, Freight Display, Garamond, GT Alpina). Headlines are light weight, generous tracking, generous leading. Body is small, airy, never dense. Type never fights for attention — it guides.
**Color:** Near-neutral. Ivory, warm white, deep charcoal, muted gold or sage as accents. Never pure white (#fff) or pure black (#000) — always a tone.
**Layout:** Maximum whitespace. Elements placed sparingly. Asymmetric but never chaotic. Margins are generous. Nothing is crammed.
**Imagery:** Exceptional photography. Warm tones, natural light, texture. Never stock. If no photography exists, the design must work without it.
**Avoid:** Bold type, bright colors, busy layouts, icons, badges, banners, anything that feels promotional or loud.

---

### 5. Art Deco / Geometric
**Emotion:** Drama, glamour, confidence, sophistication
**Best for:** Events, entertainment, bars, restaurants, fashion, beauty, hospitality

**Type:** High-contrast display serifs or geometric sans. Headlines are tall and bold. Gold or deep color type on dark backgrounds or vice versa. Decorative caps acceptable.
**Color:** Deep, saturated backgrounds — navy, black, forest, burgundy. Metallic accents: gold, brass, copper. Cream or ivory as text.
**Layout:** Symmetry is used intentionally. Geometric shapes as structural elements. Borders, lines, and ornamental details with purpose.
**Imagery:** High-contrast photography. Styled, intentional. Geometric crops or frames.
**Avoid:** Flat modern color palettes, thin sans-serifs, airy whitespace, anything that reads as contemporary tech.

---

### 6. Japanese Minimalism (Ma)
**Emotion:** Calm, trust, presence, mindfulness
**Best for:** Wellness, meditation, tea, ceramics, architecture, high-end food, slow lifestyle brands

**Type:** Often mixing a refined Latin serif with Japanese-influenced spacing principles. Extreme restraint — very little type, very large leading, generous margins. Nothing competes.
**Color:** Derived from nature. Warm grays, pale greens, blush, stone, ink black. Never bright. Palette is usually 3 colors maximum including off-white.
**Layout:** Ma (negative space) is the point. Elements don't fill the page — they inhabit it. One element per visual field. Asymmetric, but perfectly balanced.
**Imagery:** Natural, textural, quiet. Macro photography. Imperfection is intentional (wabi-sabi). Never glossy or perfectly staged.
**Avoid:** Busyness, multiple competing elements, bright color, grid rigidity, anything that demands attention rather than earning it.

---

### 7. Mid-Century Modern
**Emotion:** Warmth, optimism, craftsmanship, character
**Best for:** Furniture, interiors, coffee, independent retail, craft brands, local businesses with personality

**Type:** Geometric sans-serifs (Futura, Brandon Grotesque, Circular) or slab serifs with character. Warm, friendly. Bold but not aggressive.
**Color:** Warm and earthy — terracotta, mustard, olive, cream, burnt orange, teal. Never cold. Palette feels hand-selected, not digital.
**Layout:** Organic and approachable. Not rigidly geometric — elements have breathing room but feel cozy. Illustration integrates with layout.
**Imagery:** Warm-toned photography. Product-focused. Lifestyle, but real. Or period-influenced illustration.
**Avoid:** Cold colors, rigid Swiss grid, minimal aesthetic, anything that feels corporate or tech.

---

### 8. Organic / Earthy
**Emotion:** Trust, authenticity, groundedness, sustainability
**Best for:** Food brands, farms, wellness, sustainable products, natural beauty, community organizations

**Type:** Humanist serifs or hand-influenced sans-serifs. Warm, approachable. Never geometric or cold. Body text is readable and generous.
**Color:** Natural pigments — soil, leaf, stone, sky, bark. Muted, never neon. Palette feels like it came from the ground.
**Layout:** Flows, doesn't snap to grid rigidly. Rounded sections, organic shapes as dividers. Generous whitespace that breathes.
**Imagery:** Real people, real places, natural textures. Grain is acceptable. Never polished to the point of unreality.
**Avoid:** High contrast, sharp geometry, cold grays, anything that reads as industrial or tech.

---

### 9. Dark / Tech
**Emotion:** Power, focus, sophistication, forward-motion
**Best for:** SaaS, developer tools, AI products, security, gaming, crypto, performance products

**Type:** Clean, legible sans-serif. Monospace for code or technical elements. High contrast white on dark. Headlines can be bold and geometric.
**Color:** Dark backgrounds (not pure black — dark navy, charcoal, near-black with a hue). Bright accent: electric blue, cyan, green, purple. Used sparingly and with precision.
**Layout:** Dense but organized. Grid-tight. Data and UI elements are part of the design language. Dark mode is not just an inversion — it's the primary aesthetic.
**Imagery:** Abstract, data visualization, UI screenshots, or dramatic product photography. Blue-toned. High contrast.
**Avoid:** Warm colors, organic shapes, serif type, anything that reads as soft, casual, or lifestyle.

---

### 10. Maximalist / Eclectic
**Emotion:** Joy, energy, personality, surprise
**Best for:** Youth brands, music, cultural events, bold consumer products, creative studios that want to stand out

**Type:** Multiple typefaces used intentionally — each with a job. Size variation is extreme. Color type on color backgrounds. Nothing is timid.
**Color:** Many colors, but with rules. Often a dominant color with 3–4 supporting. Contrast is high throughout. Nothing recedes.
**Layout:** Full, layered, surprising. Elements overlap. Patterns as backgrounds. Sections feel different from each other. Scroll reveals.
**Imagery:** Illustration, photography, and graphic elements coexist. Collage is valid. Texture is present.
**Avoid:** Restraint for its own sake, uniform spacing, a single neutral palette, anything that disappears.

## Typography as Foundation

Type is the first design decision — not the last. Most AI slop fails here because it picks a font for legibility rather than character, then applies it uniformly without a system.

**Rules:**

**Use max 2 typefaces.** One for display/headlines, one for body. If a single typeface has enough weight variation, use one. Never mix 3+.

**Establish a type scale before designing.** Don't improvise sizes. A scale creates rhythm. Example (8px base):
- Display: 72–96px
- H1: 48–64px
- H2: 32–40px
- H3: 24–28px
- Body: 16–18px
- Small/Caption: 12–14px

**Use weight contrast dramatically.** A 900-weight headline next to a 300-weight subheadline creates tension and hierarchy. Uniform weight creates mush.

**Leading (line-height) sets the mood.** Tight leading (1.1–1.2) is editorial, urgent, bold. Loose leading (1.6–1.8) is airy, premium, calm. Match it to the aesthetic.

**Letter-spacing serves the weight.** Tightly-spaced headlines feel urgent and bold. Widely-tracked headlines in light weight feel refined and expensive. Never track body text wider than +20.

**Never pick a font because it's "clean" or "modern."** Pick it because it carries the right character. Inter is not a design decision — it's the absence of one.

**Font pairings by aesthetic:**
- Swiss: Helvetica Neue / Aktiv Grotesk / Neue Haas Grotesk
- Editorial: Canela + Söhne / Freight Display + Neue Haas Grotesk
- Brutalist: Obviously / Dharma Gothic / Space Grotesk pushed hard
- Luxury Minimal: GT Alpina / Canela / Garamond
- Art Deco: Quiche / Cormorant / Editorial New
- Mid-Century: Futura / Brandon Grotesque / Playfair Display
- Organic: Lora / Recoleta / DM Serif Display
- Dark/Tech: IBM Plex / Space Grotesk / Söhne Mono
- Maximalist: Any, but with intentional contrast and hierarchy

## Color as Structure, Not Decoration

The default AI color approach: white background, dark text, one accent color on buttons. This is not a color system — it's the absence of one.

**Every color has a job. Assign roles before picking values:**

| Role | Job | How many |
|------|-----|----------|
| Background | The world the content lives in | 1–2 |
| Foreground | Primary text and content | 1 |
| Accent | Draws the eye, signals action | 1 (max 2) |
| Signal | CTAs, alerts, interactive elements | 1 |
| Neutral | Supporting, receding elements | 1–2 |

**Rules:**
- Never use color just to fill space
- The accent color must earn its use — if it appears everywhere, it loses meaning
- Contrast is a conversion tool: the CTA should have the highest contrast relationship on the page
- Background is not always white — a warm cream, a deep navy, a soft sage can define the entire aesthetic
- Never use pure #000000 black or pure #ffffff white — add a hue (warm or cool) to both
- Test every color combination for WCAG AA contrast (4.5:1 for body text, 3:1 for large text)

## Spatial Systems

Random spacing is invisible but felt. Consistent spacing creates rhythm. Inconsistent spacing creates unease.

**Rules:**
- Use an 8px base unit. All spacing values are multiples: 8, 16, 24, 32, 48, 64, 96, 128
- Section padding is not uniform — vary it intentionally to create rhythm and differentiate sections
- Whitespace is not empty — it is structural. Generous margins signal quality and confidence
- Align to a column grid. 12-column is standard. Never free-float elements
- Intentional asymmetry (a 7-column/5-column split) is more interesting than 6/6 every time
- Mobile spacing: reduce by ~30–40%, but never collapse to zero. Whitespace on mobile is still valuable

## AI Slop Anti-Patterns

These patterns must be actively avoided. They are the signature of default AI output. If any of these appear in a design direction, revise before delivering.

**Layout:**
- ❌ Hero → 3 equal-width cards → testimonials → CTA banner → footer (the default layout)
- ❌ Everything centered on the page
- ❌ All sections the same height with the same padding
- ❌ Features displayed as icon + title + 2 sentence description in a 3-column grid

**Visual:**
- ❌ Purple or blue gradient on the hero section
- ❌ All-white (#fff) background with a single accent color only appearing on buttons
- ❌ Box shadow (0 4px 6px rgba(0,0,0,0.1)) on every card
- ❌ border-radius: 9999px on every element (pill shapes everywhere)
- ❌ Stock photo with semi-transparent dark overlay and white text on top
- ❌ Decorative blobs or circles as background shapes

**Typography:**
- ❌ Inter (or system-ui) for everything — headline and body
- ❌ All type the same weight — no contrast
- ❌ Headline font-size only slightly larger than body (not enough contrast)
- ❌ Tight letter-spacing on light-weight body text

**Icons and Imagery:**
- ❌ Heroicons, Font Awesome, or Feather icons as the primary visual element on feature sections
- ❌ Flat vector illustrations in the same style as every other SaaS site
- ❌ Unsplash stock photography of smiling people at laptops

**Color:**
- ❌ Blue (#3B82F6) or purple (#8B5CF6) as the default accent color
- ❌ Using color only on interactive elements — nowhere else
- ❌ 5+ colors with no assigned roles

## Conversion Principles

Distinctive design that doesn't convert is decoration. Both matter.

**Visual hierarchy drives conversion:**
The eye must arrive at the primary CTA without effort. Create a clear path: headline → supporting detail → CTA. Everything else is secondary.

**Contrast = priority.** The highest-contrast element on the page gets looked at first. The CTA should always be one of the 2–3 highest-contrast elements. If everything has equal contrast, nothing gets clicked.

**One primary CTA.** Secondary actions exist but never compete visually. The primary CTA is the most prominent button. Never style two buttons identically.

**Above the fold:** The user must understand what the business does and see the primary CTA without scrolling. This is non-negotiable.

**Reading patterns:**
- **F-pattern:** Users scan down the left side, then horizontally on headlines/subheadlines. Put the most important content left and top.
- **Z-pattern:** On sparse pages, the eye moves top-left → top-right → diagonal → bottom-right. Logo top-left, CTA top-right, conversion bottom-right.

**Trust signals reduce friction.** Place near CTAs: testimonials, client logos, number stats, guarantees, certifications. Not in a dedicated section at the bottom — adjacent to the moment of decision.

**Friction is invisible.** Long forms, vague CTAs ("Submit"), no social proof, too many choices, slow page load — all reduce conversion without the user knowing why. Audit for friction, not just aesthetics.

## Self-Critique Loop (Mandatory)

Before finalizing any design direction or wireframe brief, score it against these four criteria. If any score is below 3, revise before delivering.

**Score each 1–5:**

| Criterion | Question | Low (1–2) | High (4–5) |
|-----------|----------|-----------|------------|
| **Coherence** | Does it feel like a whole? | Parts feel borrowed from different designs | Every element feels like it belongs to the same world |
| **Originality** | Are there deliberate non-default choices? | Could have been generated by default settings | Clear creative decisions that wouldn't happen by accident |
| **Craft** | Is the typography, spacing, and color technically sound? | Improvised type, random spacing, no color logic | Established type scale, spatial system, color roles assigned |
| **Conversion** | Can the user find the primary action in 3 seconds? | Unclear hierarchy, competing CTAs, CTA buried | Clear path from entry to action, CTA is the most prominent element |

**If score is low:**
- **Coherence < 3:** Return to the aesthetic library. Pick one and apply its rules consistently. Remove anything that doesn't belong to that aesthetic.
- **Originality < 3:** Check the anti-patterns list. Replace every default choice with an intentional one. Ask: would this layout exist without AI generation?
- **Craft < 3:** Define the type scale explicitly. Assign spacing values to the 8px system. Name every color's role before using it.
- **Conversion < 3:** Identify the primary CTA. Make it the highest-contrast element on the page. Ensure it's visible above the fold. Remove anything competing with it.

## How to Use This Skill

1. **At intake:** Run Step 0 — answer all 4 questions before any design work
2. **When choosing aesthetic:** Consult the Aesthetic Library — pick one, name it, follow its rules
3. **When planning type:** Use the Typography section — establish scale, pick a pairing, use weight contrast
4. **When choosing color:** Assign roles before values — never add color without a job
5. **When planning layout:** Apply the Spatial System — 8px unit, column grid, vary section rhythm
6. **Before delivering:** Run the Self-Critique Loop — score all 4 criteria, revise anything below 3
7. **At review:** Check against the Anti-Patterns list — eliminate any slop patterns found
